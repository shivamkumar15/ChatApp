
# ChatApp 

Welcome to **ChatApp** — a real-time chat website built using **JavaScript** and **ChatEngine**!

This project lets users connect and chat instantly with others through a modern, responsive web interface. It uses the **ChatEngine API** to handle real-time messaging and communication across clients.

---

##  Features

 Real-time messaging  
 Multiple user support  
 Simple UI built with HTML, CSS & JavaScript  
 Powered by ChatEngine for seamless real-time communication  
 Easy to configure and extend

---

##  Built With

 **JavaScript** — Core language for frontend logic  
 **HTML & CSS** — Markup and styling
